# renpy-localize

A lightweight, offline Ren'Py translation tool powered by [Ollama](https://ollama.com).
Translates untranslated `.rpy` files using a local LLM — no API key, no cloud, no cost.

## Features

- **Fully offline** — runs entirely on your machine via Ollama
- **Zero dependencies** — pure Python standard library, no `pip install` needed
- **Tag protection** — Ren'Py tags (`{...}`, `[...]`) are masked during translation and restored afterwards, with automatic validation and retry
- **Context-aware** — passes preceding dialogue lines to the LLM for consistent translations
- **Batch translation** — translates multiple entries per API call for efficiency
- **Interrupt-safe** — saves partial progress on Ctrl+C or connection error
- **Auto backup** — creates timestamped backups before modifying any file
- **Force re-translate** — re-translate already translated entries with `--force`
- **Dry run** — preview translations without writing to files

## Requirements

- Python 3.10+
- [Ollama](https://ollama.com) running locally

## Setup

1. Install and start Ollama:
   ```bash
   ollama serve
   ollama pull qwen3:14b
   ```

2. Clone this repo:
   ```bash
   git clone https://github.com/ByLightLeak/renpy-localize.git
   cd renpy-localize
   ```

3. (Optional) Edit `config.json` to change the model or settings.

## Usage

```bash
# Translate a single file
python translate.py path/to/translation.rpy

# Translate all .rpy files in a directory
python translate.py path/to/tl/chinese/

# Recursively search subdirectories
python translate.py path/to/game/ -r

# Preview without writing
python translate.py path/to/translation.rpy --dry-run

# Re-translate already translated entries
python translate.py path/to/translation.rpy --force

# Use a different model
python translate.py path/to/translation.rpy --model qwen3:8b
```

## Configuration

Edit `config.json` to set your defaults:

```json
{
  "ollama_host": "http://localhost:11434",
  "model": "qwen3:14b",
  "batch_size": 5,
  "context_lines": 5,
  "num_ctx": 8192
}
```

| Key | Description |
|-----|-------------|
| `ollama_host` | Ollama API URL |
| `model` | Ollama model name |
| `batch_size` | Entries per API call (1–20) |
| `context_lines` | Preceding dialogue lines passed as context |
| `num_ctx` | Ollama context window size (tokens) |

## Notes

- Currently translates **English → Traditional Chinese (繁體中文)**
- Designed for Ren'Py translation files generated by `renpy translate`
- Tested with `qwen3:14b`; other models may work with varying quality

## 中文說明

輕量的離線 Ren'Py 翻譯工具，透過 [Ollama](https://ollama.com) 在本地執行 LLM 進行翻譯。
不需要 API 金鑰、不上雲端、完全免費。

### 特色

- **完全離線** — 透過 Ollama 在本機執行，不需網路
- **零依賴** — 只需 Python 標準函式庫，不需 `pip install`
- **Tag 保護** — 翻譯前自動遮蔽 Ren'Py tag（`{...}`、`[...]`），翻譯後還原並驗證，失敗自動重試
- **上下文感知** — 將前幾行對話傳給 LLM，確保翻譯前後一致
- **批次翻譯** — 每次 API 呼叫翻譯多筆，提升效率
- **中斷保護** — Ctrl+C 或連線失敗時自動儲存已完成的進度
- **自動備份** — 修改檔案前建立含時間戳的備份
- **強制重譯** — 用 `--force` 重新翻譯已有譯文的條目
- **預覽模式** — 用 `--dry-run` 預覽翻譯結果，不寫入檔案

### 需求

- Python 3.10+
- 本地執行中的 [Ollama](https://ollama.com)

### 安裝

1. 安裝並啟動 Ollama：
   ```bash
   ollama serve
   ollama pull qwen3:14b
   ```

2. Clone 這個 repo：
   ```bash
   git clone https://github.com/ByLightLeak/renpy-localize.git
   cd renpy-localize
   ```

3. （選用）編輯 `config.json` 調整模型或設定。

### 使用方式

```bash
# 翻譯單一檔案

# 翻譯資料夾內所有 .rpy 檔案

# 遞迴搜尋子資料夾
python translate.py path/to/game/ -r

# 預覽翻譯結果（不寫入）
python translate.py path/to/translation.rpy --dry-run

# 強制重新翻譯已有譯文的條目
python translate.py path/to/translation.rpy --force

# 使用不同模型
python translate.py path/to/translation.rpy --model qwen3:8b
```

### 設定

編輯 `config.json`：

```json
{
  "ollama_host": "http://localhost:11434",
  "model": "qwen3:14b",
  "batch_size": 5,
```

| 參數 | 說明 |
